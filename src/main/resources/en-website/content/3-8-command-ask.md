title=Ask Command
title_en=ask
~~~~~~
parameters:
person - person’s username

This command is only to be used by [detector](#detector) and at night while Mafia kills someone, then its detectors turn to ask the Bot privately about a suspicious user whether he/she is a mafia or not. e.g. if the detector is suspicious about “Robert”, to ask it from the bot, he/she enter the following command privately to the bot:
<pre><code>/ask Robert</code></pre>
