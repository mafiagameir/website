title=Heal Command
title_en=heal
~~~~~~
Parameters:
Person - the username of person

this command is solely for the [Doctor](#doctor) and at night, after mafia killed a player, the doctor will be asked who he/she wants to be healed. Like [kill command](#kill), this command should also being executed privately in a message to the Bot and not in the public group.
e.g. the Doctor should enter the following command to the Bot in a private chat to heal the user “Alice”:
<pre><code>/heal Alice</code></pre>
